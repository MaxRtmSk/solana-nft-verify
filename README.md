# SOLANA NFY VERIFY

## How start emulator

1. git clone this repo
2. go to project folder > functions
3. start `npm i`
4. install firebase-tools `npm install -g firebase-tools`
5. start `npm run serve`

## Open Endpoints

Open endpoints require.

- [VerifyNft](verifyNft.md) : `GET /verifyNft/`
